Removes the email field from WordPress comments form.

== Description ==

Removes the email field from WordPress comments form. You should use this plugin because you don't want to store this piece of data (email) into your database, which might identify some people. This simple snippet may be of special interest for those willing to comply with Spanish law LOPD.

Quita el campo email del formulario de comentarios de WordPress. Puedes utilizar este plugin porque no quieres almacenar el email de los usuarios en tu base de datos, lo que podría identificar a algunas personas, de acuerdo a la ley española. Esto es de especial interés para aquellos que quieren cumplir con la Ley Orgánica de Protección de Datos LOPD.

== Installation ==

1. Upload the `remove-email-from-comments` folder to your `/wp-content/plugins/` directory.

2. Activate the plugin in your WordPress administration interface.
